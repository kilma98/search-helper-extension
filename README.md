# 🔍 Search Helper

A lightweight browser extension that enhances your search queries with smart suggestions.

Instead of guessing how to search better, Search Helper generates optimized queries using advanced search operators — making research faster and more effective.

---

## ✨ Features

- 🔎 Generate smarter search queries instantly
- 🎯 Multiple modes:
  - General
  - Academic
  - Technical
- 📄 Find PDFs, reports, and structured content
- 🧠 Learn *why* each query works (built-in explanations)
- 🚀 Open one or multiple searches in one click
- 📋 Copy queries easily

---

## 🧰 Tech Stack

- TypeScript
- Vite
- Chrome Extension (Manifest V3)
